# Site estático

Site HTML/CSS/JS puro, pronto para deploy no Vercel.

## Deploy no Vercel

1. Suba o repositório no GitHub.
2. No Vercel: **Add New Project → Import** o repositório.
3. Framework Preset: **Other** (deixe Build Command e Output Directory em branco).
4. Clique em **Deploy**. Pronto.

Não requer `npm install` — é 100% estático.
